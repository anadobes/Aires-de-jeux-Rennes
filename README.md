# Aires-de-jeux-Rennes

- **Aires.csv** : Fichier contenant la liste des aires de jeux de Rennes
- **app.html** : Page principale de l'application
- **index.php** : Page qui génère le JSON
- **insert.php** : Page utilisée pour insérer les données du CSV en BDD, pourra être exécutée régulièrement via CRON
- **js/script.js** : Javascript exécutant l'application
